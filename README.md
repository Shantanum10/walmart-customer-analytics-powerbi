# 🛒 Walmart Customer Analytics Dashboard

An end-to-end Power BI project that analyzes Walmart customer data to uncover insights into customer behavior, churn, customer lifetime value (CLV), loyalty program performance, promotion effectiveness, and store performance. The dashboard transforms raw customer data into interactive visualizations to support data-driven business decisions.

---

## 📊 Project Overview

This project integrates **five interconnected datasets**:

- Customer Transactions
- Customer Demographics
- Loyalty Program
- Store Locations
- Churn Labelled Customers

The data model was built using **Customer_ID** and **Store_ID** relationships to create a unified analytical model for customer-centric reporting.

---

## ⚙️ Data Preparation

The project includes multiple calculated columns and DAX measures to enhance business analysis.

### Calculated Columns
- Membership Duration
- Customer Lifetime Value (CLV)
- CLV Segment
- Days Since Last Purchase
- Customer Segment
- Age Group

### DAX Measures
- Churn Rate %
- Retention Rate
- Repeat Rate
- Total Customers
- Churned Customers
- Average Purchase Frequency
- Average Purchase Amount
- Average Transaction Amount
- Promotion Transaction %
- Total Points Earned
- Total Points Redeemed
- Average Customer Lifetime Value (CLV)

---

## 📈 Dashboard Features

The solution consists of **4 interactive dashboard pages**, each designed to answer specific business questions.

### 1. KPI Overview
- Total Customers
- Churn Rate
- Average Customer Lifetime Value
- Repeat Rate

### 2. Loyalty & Promotion Impact
- Promotion Transaction %
- Average Purchase Amount
- Churn Rate by Loyalty Tier
- Points Earned vs Points Redeemed

### 3. Store & Channel Insights
- Average Transaction Amount by Store Type
- Churn Rate by Store Type
- Store Opening Year vs Retention Rate

### 4. Customer Segmentation
- Customer Lifetime Value Analysis
- High vs Low CLV Customers
- Customer Inactivity Analysis
- Loyalty Tier Analysis
- Regional Customer Analysis

---

## 💡 Key Business Insights

- Identified high-value customers using Customer Lifetime Value (CLV).
- Measured customer churn and retention trends.
- Evaluated the effectiveness of promotional campaigns.
- Analyzed loyalty program engagement through reward points earned and redeemed.
- Compared store performance using transaction value and retention metrics.
- Segmented customers based on purchasing behavior to support targeted marketing strategies. :contentReference[oaicite:3]{index=3}

---

## 📌 Business Recommendations

- Re-engage inactive customers with high Customer Lifetime Value.
- Target low-tier customers with personalized promotions to improve retention.
- Improve loyalty program participation by simplifying reward redemption.
- Strengthen customer experience across underperforming sales channels.
